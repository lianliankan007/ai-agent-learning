python语法学习

1.self这python语法中类似Java的this
关键点总结
要点	                说明
必须作为第一个参数	       实例方法的第一个参数必须是 self
调用时不用传	          agent.chat("你好") 自动把 agent 作为 self 传入
区分不同实例	          每个实例有自己的 self，数据互不干扰
访问实例属性	          self.属性名 访问当前实例的数据
调用其他方法	          self.方法名() 调用当前实例的其他方法

2._init__ Python的构造函数
def _(self, name, age):
        self.name = name
        self.age = age
